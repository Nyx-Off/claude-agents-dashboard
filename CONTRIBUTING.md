# Contributing to Claude Agents Dashboard

Thanks for your interest in contributing! This project is intentionally small and simple, and we'd like to keep it that way.

## How to Contribute

1. **Fork** the repository.
2. **Create a branch** for your change: `git checkout -b my-feature`
3. **Make your changes** and test them locally (`npm start`, then check http://localhost:8787).
4. **Commit** with a clear message describing what you changed and why.
5. **Open a Pull Request** against `main`.

## Code Style

- Keep it simple. No frameworks, no build tools, no transpilers.
- Avoid unnecessary comments -- let the code speak for itself.
- Preserve the 8-bit retro aesthetic in any UI changes.
- Test your changes manually before submitting.

## Reporting Issues

Open a GitHub issue with a clear description of the problem and steps to reproduce it.

## Questions?

Open an issue or start a discussion. We're happy to help.
